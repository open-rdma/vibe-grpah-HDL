# RTL Blueprint 操作手册

## 界面布局

```
┌──────────────────────────────────────────────────────────┐
│  工具栏 (Toolbar)                                        │
├──────────┬────────────────────────┬──────────────────────┤
│ 项目树   │                      │  属性面板             │
│ (左侧)   │     画布 (Canvas)     │  (右侧)              │
│          │                      │                      │
│          │                      │                      │
└──────────┴────────────────────────┴──────────────────────┘
│  状态栏 (Statusbar)                                      │
└──────────────────────────────────────────────────────────┘
```

- **左侧面板** — 项目文件树，列出所有图和目录
- **中间画布** — 节点编辑器，拖拽、连线都在这里完成
- **右侧面板** — 属性编辑器，根据选中内容显示图属性、模块属性或端口属性
- **底部状态栏** — 显示当前图路径、缩放比例、选中数量

---

## 一、创建项目

1. 点击工具栏 **New** 按钮
2. 输入项目路径（如 `my_project`）和项目名称
3. 点击 **Create**

创建后左侧会显示项目树，包含一个默认的顶层目录。

---

## 二、创建子模块

子模块就是一个 RTL 模块实例，在画布上以节点形式出现。

### 方式 A：从项目树拖拽（推荐）

1. 先在左侧项目树中创建好子模块对应的图文件（右键目录 → **New Graph**，输入名称）
2. 打开该图，在属性面板中定义好端口（见第三节）
3. 回到父级图，从左侧项目树**拖拽该图文件**到画布上
4. 节点自动创建，引用路径和端口列表自动加载

### 方式 B：工具栏添加空白节点

1. 点击工具栏 **Add Subgraph** 按钮
2. 画布上出现一个名为 `new_module` 的空白节点
3. 在右侧属性面板的 **Ref Path** 字段中手动填写引用的图路径（如 `tree_name/sub_module.yaml`）

### 方式 C：右键菜单

选中文档树中的图文件 → 右键 → **New Graph** 创建新的子图。

---

## 三、定义端口

端口是图的对外接口，分为 **图级端口** 和 **节点端口**。

### 图级端口（当前图的对外接口）

1. 点击画布空白区域，右侧显示 **Graph Info** 面板
2. 在 **Ports** 区域点击 **+ Add Port**（或首次使用点击 **+ Add First Port**）
3. 点击端口名称进入编辑：
   - **Name** — 端口名称
   - **Direction** — `input`（输入）或 `output`（输出）
   - **Category** — 信号类别：
     - `data` — 数据信号（默认）
     - `clock` — 时钟信号
     - `reset` — 复位信号
   - **Type** — 数据类型，可从下拉列表选择或自由输入
   - **Clock Domain** — 所属时钟域（如 `clk`、`clk_fast`）
   - **Reset Domain** — 所属复位域
   - **Reset Type** — 复位类型（仅 reset 类别可见）：`async` / `sync`
   - **Allow Cross-Domain Connection** — 允许跨时钟域连接（仅 data 且有 clock_domain 时可见）

4. 点击 **← Back to Graph Properties** 返回

图级端口会自动同步到画布上的边界节点（`<- graph_input` 和 `graph_output ->`）。

### 节点端口（模块实例的端口）

节点端口由引用的子图自动决定。当你设置节点的 **Ref Path** 后，系统会自动从引用的图中读取端口列表并显示在节点上。

## 四、连线

### 基本连线操作

1. 将鼠标悬停在节点的**输出端口**（右侧圆点）上
2. 按住鼠标左键拖动，会拖出一条连线
3. 将连线拖到目标节点的**输入端口**（左侧圆点）上
4. 松开鼠标完成连接

### 删除连线

- 选中连线（点击连线），按 **Delete** 键删除

### 连线规则

系统会自动校验连线是否合法，以下情况会被阻止：

| 规则 | 说明 |
|------|------|
| data → clock | ❌ 数据信号不能连接到时钟端口 |
| data → reset | ❌ 数据信号不能连接到复位端口 |
| 类型不匹配 | ❌ `logic[7:0]` 和 `logic[15:0]` 不能直连 |
| 跨时钟域 | ❌ `clk` 域的端口不能直接连 `clk_fast` 域的端口（除非任一端口启用了 Allow Cross-Domain） |

连线被阻止时，右下角会弹出红色提示说明原因。

## 五、编辑模块属性

1. **单击**画布上的模块节点选中它
2. 右侧属性面板显示：
   - **Name** — 节点名称，同一层级不可重名
   - **Description** — 描述文字
   - **Test Method** — 测试方法
   - **Ref Path** — 引用的子图路径
   - **Properties** — 自定义属性键值对，点击 **+ Add Property** 添加
   - **Ports** — 当前节点的端口列表（只读，由引用图决定），点击可查看端口详情

## 六、导航

| 操作 | 方式 |
|------|------|
| 平移画布 | 鼠标中键拖拽 或 空白区域左键拖拽 |
| 缩放 | 鼠标滚轮 |
| 放大 | 工具栏 **Zoom In** |
| 缩小 | 工具栏 **Zoom Out** |
| 适应窗口 | 工具栏 **Fit** |
| 进入子图 | 双击模块节点，或右键 → **Open Subgraph** |

---

## 七、保存与构建

- **Save** — 保存当前图到文件
- **Build** — 构建当前模块，弹出对话框配置：
  - Scope：构建范围（仅当前 / 含子模块 / 含父模块 / 全图）
  - Mode：Fresh（全新生成）或 Incremental（增量更新）
  - Include testbench：是否包含测试平台

---

## 八、类型系统

点击工具栏 **Types** 按钮打开类型编辑器：
- 查看已定义的数据类型
- 添加新类型（名称 + 描述）
- 删除不需要的类型
- 点击 **Save** 保存到服务器

类型会在端口编辑的 Type 字段中以下拉列表形式出现。

---

## 九、快捷操作速查

| 操作 | 方法 |
|------|------|
| 创建图 | 左侧项目树右键目录 → New Graph |
| 创建目录 | 左侧项目树右键 → New Tree |
| 添加模块 | 拖拽图文件到画布 或 工具栏 Add Subgraph |
| 添加端口 | 点击空白处 → Ports → + Add Port |
| 连线 | 从输出端口拖到输入端口 |
| 删除节点/连线 | 选中后按 Delete |
| 删除端口 | 点击端口名进入编辑 → 点击 ✕ 按钮 |
| 重命名节点 | 选中节点 → 右侧 Name 字段修改 → 回车确认 |
| 查看子图 | 双击模块节点 |
