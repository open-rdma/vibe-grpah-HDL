## Module Description:
A collection of primitive utility functions and interfaces used throughout the RDMA design. This module provides:
1. Bit manipulation functions: isZero (check if all bits are zero),
   isZeroR (recursive divide-and-conquer version of isZero),
   isLessOrEqOne (check if value is 0 or 1), isOne (check if value is 1),
   isTwo (check if value is 2), isLargerThanOne (check if value > 1),
   isAllOnesR (check if all bits are 1 using recursive approach),
   isZeroByteEn (check if byte enable is zero).

2. A specialized function isZero4LargeBits for checking two halves of a
   32-to-64-bit value separately, using provisos to constrain the size.

3. Bit manipulation helpers: zeroExtendLSB (extend a value by adding zero
   bits at the least significant end), removeMSB (drop the most significant
   bit), dontCareValue (return a don't-care value using ?), unwrapMaybe
   (extract value from Maybe with don't-care default), unwrapMaybeWithDefault
   (extract value from Maybe with explicit default).

4. Tuple accessor functions: getTupleFirst through getTupleSixth for
   extracting elements from 2-to-6-element tuples.

5. identityFunc: returns its input unchanged.
6. Assertion helpers: immAssert (immediate assertion check -- if condition
   is false, print error with position info and finish simulation),
   immFail (unconditional failure with message).

7. PipeOut conversion: toPipeOut converts a FIFOF to a PipeOut interface.
8. FlagsType system: a generic struct that wraps an enum's bit representation
   for use as a flags bitmask. Includes FShow instance, enum2Flag conversion,
   containFlags (check if one flags value contains another), containEnum
   (check if a flags value has a specific enum bit set).

9. CountCF interface and mkCountCF module: a conflict-free counter that
   supports concurrent increment, decrement, write, and read operations.
   The increment and decrement operations are enqueued into FIFOs and
   processed in rules. A write operation clears pending incr/decr.
   Uses mkCReg for safe concurrent register access.

This module imports FIFOF and PAClib. It depends on no other project modules.
